# Smart cooking app

## Git clone

    git clone https://github.com/Devkeystuff/smart-cooking.git

## Installation

### Node

Use node v17.0.x

https://nodejs.org/en/

## Python

Use python v3.10.x

## Justfile
For project management we're using `Justfile`

    cargo install just

Install dependencies

    just setup

Run project

    just dev
