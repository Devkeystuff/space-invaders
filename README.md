### Node

Use node v17.0.x

https://nodejs.org/en/

Use python v3.10.x

## React native

Use react-native-cli v2.0.x

## Justfile
For project management we're using `Justfile`

### For windows:

- Install [WSL](https://docs.microsoft.com/en-us/windows/wsl/install)

- Activate WSL
    
        wsl

    cargo install just

Install dependencies

    just setup

Run project

    just dev
