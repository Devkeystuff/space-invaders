## Installation

### Python

Use python v3.10.x

#### Black

For formatting and installing dependencies

##### Installation

    pip install black

## Actions

For code formatting and more concise dependency installation we're using `Makefile`

### Install dependencies
    
    make setup
    
### Start project

    make dev
    
### Format code
    
    make format
